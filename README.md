## 느낀점
 흔히 말하는 마개조를 처음부터 내 입맛에 맞게 제작하느라 제작 속도가 안 나 모든 도전 기능을 다 구현 못 한 것이 아쉽다. 추가로 가져온 3D에셋에 이미 이동 관련 로직이 있어 학습한 방식과 다르 게 진행이 되어 학습영상 내용을 그대로 못 쓴 점이 아쉽다.
